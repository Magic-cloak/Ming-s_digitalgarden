# 1.皮蛋擂茄子

### 材料准备

- [ ] 条茄子---2根
- [ ] 青椒---4个
- [ ] 皮蛋---4个
- [ ] 蒜---5瓣
- [ ] 生抽---2小勺
- [ ] 老抽---0.5小勺
- [ ] 蚝油---1小勺
- [ ] 醋---适量

## 制作步骤

1. 茄子按条切开，蒸到可以用筷子戳透（约20min）
2. 青椒干煸到表面焦黄，烧黑的部分表皮撕皮去掉
3. 蒜瓣捣成蒜泥，不需要捣特别碎
4. 皮蛋去壳
5. 茄子、青椒按纤维走向切成条状，皮蛋切开，方便装入容器捣
6. 食材全部装入容器，捣匀
7. 加入生抽、老抽、蚝油、醋、适量盐，拌匀
